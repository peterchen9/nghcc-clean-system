# 清潔工作系統

手機友善的清潔人員工作系統原型，可直接用瀏覽器開啟 `index.html` 執行。

## 功能

- 後台新增、停用、刪除工作人員
- 後台新增、刪除工作區域
- 後台依「星期 / 人員 / 區域」安排打掃內容
- 工作人員登入後可看到自己的打掃區域
- 工作人員可上班、下班打卡
- 工作人員可看到自己未結算工時與預估工資
- 後台可依選定日期結算薪資，結算後該區間工時不再計入未結算
- 後台可做 1-5 分清潔檢查
- 檢查低於 3 分時，該人員下次查看同一區域會以紅字顯示「請加強清潔」

## 使用方式

1. 開啟 `index.html`
2. 工作人員可直接選姓名登入
3. 後台登入密碼預設為 `admin123`

## 目前資料儲存方式

資料儲存在瀏覽器的 `localStorage`，適合單機展示或流程確認。若要讓多位人員在不同手機共用即時資料，下一步應改成後端資料庫與帳號登入。
